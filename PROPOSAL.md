# Mini Project 3 Proposal

## What I'm building
A book search web app that allows users to search for books and view details like title, author, and cover image.

## Which API I'm using
Open Library API  
https://openlibrary.org/developers/api

## Why I chose this
I chose this API because I enjoy books and wanted to build something interactive and useful. The API provides a lot of data like titles, authors, and cover images, which makes it great for building a searchable interface.

## Core features
- Search for books by title or keyword
- Display a list of books with title, author, and cover image
- Show a loading state while fetching data
- Show an error or “no results found” message
- Click on a book to view more details

## What I don't know yet
I am still learning how to use fetch() with async/await and how to work with JSON data from APIs. I also want to improve my understanding of dynamically updating the page based on user input.
